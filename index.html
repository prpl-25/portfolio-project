Finding near-optimal solutions to TSP using reinforcement learning

This project uses an auction-based path-finding heuristic within an RL framework. It is suited to TSP, as it computes paths without enumerating all subtours. We compared it to a greedy heuristic and the optimal solution, showing that this approach effectively generates near-optimal solutions for TSP.

Funded by FURI and NSF REU award.
